# HTML-LEGO
💡 功能說明書：

🎵 樂高喀嚓聲：點擊按鈕或刪除卡片時，會有清脆的積木組裝音效。

🏎️ 動態時速模擬：時速錶現在會像真實開車一樣，在 80∼120 之間帥氣地跳動。

✨ 驚喜按鈕動畫：按下去的瞬間會有細微的震動回饋。

聽聽看！：現在妳點擊右邊零件箱的積木，或是點右上角的太陽/月亮，都會發出輕微的「喀嚓」聲。這是用數學頻率產生的音效，不用擔心要下載音效檔喔！

看錶！：時速錶現在不會死板板地停在 88 了，它會隨著時間在 80∼120 之間上下跳動，模擬真實開車的感覺。

更有動感：加了 transition-group。現在妳增加或刪除積木卡片時，旁邊的卡片會自動「滑」過去補位，動作非常流暢！
